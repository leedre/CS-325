INSTRUCTIONS

    HOW TO COMPILE  ->  tsp.cpp
        make

    HOW TO RUN  ->  tsp
        ./run-tsp.sh

    OR
        ./tsp <file-name>

        EXAMPLE
	        ./tsp test-input-1.txt
    

    HOW TO RUN  ->  tsp-verifier.py
        ./run-tsp-verifier.sh

    OR
        Follow instructions in python files


RESULTS

    Output files that contain the tour length and tour itself will have 
    the file extention of ".tour"

    Output file that contains the times for "test-input-x.txt" will have
    the file name of "test-input-times.txt"

    Output file that contains the times for "tsp_example_x.txt" will have
    the file name of "tsp_example_times.txt"
